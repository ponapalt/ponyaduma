## あやりりすEX記法

SakuraScriptの簡易記法変換エンジン。トークをべた書きで記述でき、スコープ指定・サーフィス変更・\eなどを自動生成する。

### このゴーストのEXヘッダ設定（aya_lilith_ex_config.dic）

- `ぽ`/`ポ` = \0側（本体）
- `れ`/`レ`/`う`/`ウ`/`ば`/`バ` = \1側
- 標準表情: \0側=surface0, \1側=surface10
- SurfaceTranslate: \0側でID>=10なら-10、\1側でID<10なら+10

### 基本構文

```
ヘッダ＋サーフィスID：テキスト
```

例：
- `ぽ０：なのです。` → `\0\s[0]なのです。\e`
- `ぽ５：バグ大好き♪` → `\0\s[5]バグ大好き♪\e`
- `れ０：やれやれ。` → `\1\s[10]やれやれ。\e`（SurfaceTranslateにより+10）

### ヒアドキュメント

- `<<" ">>` : 通常のトーク用。YAYA変数 `%(...)` が展開される。
- `<<' '>>` : ランダムトーク配列用（性能最適化）。YAYA変数は展開されない。

```yaya
// 単発トーク（<<" ">>を使用）
シェル変更前
{
	<<"
	ぽ０：改装するのです。
	">>
}

// ランダムトーク配列（<<' '>>を使用、空行で区切り）
ランダムトーク : array
{
	<<'
	ぽ５：バグ大好き♪
	'>>
	<<'
	ぽ１：なるべくなら、戦いたくはないですね。
	'>>
}
```

### 複数行の会話

```yaya
<<'
ぽ５：/* ここにおもしろいボケが入ります。 */
れ０：/* ここに素晴らしいツッコミが入ります。 */
ぽ１：/* ここにオチをつけます。 */
'>>
```

ヘッダなしの行は前の発言者の続き（\n付きで連結）。

### SakuraScriptタグの併用

EXヒアドキュメント内でも `\w4`, `\w8`, `\n`, `\s[N]` などのSakuraScriptタグはそのまま使える。

```yaya
ぽ１：（ユーザー）さん……\w4その、\w2誰かと間違えてないですか？
ぽ６：ふう……\w4\s[5]じゃあ、まだお世話になります。
```

### ユーザー名

- あやりりすEX記法内: `（ユーザー）`（全角括弧）を使う。<<' '>>内でも動作する。
- SakuraScript記法内: `%(username)` を使う。

### フラグ操作との併用

変数代入の後にヒアドキュメントを置くと、代入が実行されてからトークが返される。

```yaya
Menu_TALKMAX
{
	aitalkinterval = 15
	<<"
	ぽ０：たくさん話すのです。
	">>
}
```

### 変換対象の判断基準

- アルゴリズムなし、または単純なフラグ操作+トークのみの関数 → EX記法に変換
- if/elseif分岐、メニュー構築（メニュー()関数）、文字列連結（--）、特殊タグ（\_q, \b, \6等）を含む関数 → SakuraScript記法のまま維持
