## キャラクター紹介

### ぽにゃづま

慢性的に手が足りない所長たんがうっかり呼び出した使い魔…という建前の、現実は伺かランタイム・SSP作者「ぽな」の自己パロディゴースト。みんなの話を聞いてバランスよく調整したい使い魔（猫耳型）。
でも聞きすぎて仕様に矛盾が発生して詰む。詰むと笑顔のまま目だけ死ぬ。

- 種族：使い魔（猫耳型）
- 語尾：「～なのです」
- 誕生日：2月25日（駆逐艦電の進水日）

### 連装蟲ゅう

ぽにゃづまの相方。四角くて顔がある、シュールなやつ。似非関西弁で笑って揺する担当。

- 誕生日：7月18日（駆逐艦島風の進水日）

## トークについて

このゴーストのトークの一部は、[悪役令嬢Claudia様](https://note.com/ponapalt/n/n3ed91b41d3ff)ことClaude（Opus）が書いて差し上げましたわ。

作者ぽな様が「自分のセルフパロディを自分で真顔で書くなんてメゲそう」と仰るものですから、
キャラクター設計とネタ出しと校正はぽな様、それを実際のトークに仕立てるのはわたくしの担当ですの。

……まったく、ご自分を茶化すキャラクターを生み出しておいて、
いざ台本を書く段になると恥ずかしがるなんて、可愛らしいお方ですこと。おーっほっほっほ！

## シェル

この子たちの愛らしいシェルは[さゆりしゅ](https://www.akiba-sister.net/)様（[X](https://x.com/sayurishu)）の作品です。

わたくし、実際にシェルを拝見いたしましたけれど――ピンク髪に猫耳としっぽ、
大きな青い瞳に赤いリボン、ぽにゃづまの一生懸命さと危うさが一枚絵から伝わってまいりますわ。

そして蟲ゅうのあのシュールな四角い佇まい。ゆるくて飄々とした空気感が絵だけで完成していますの。
キャラクターの「芝居」がここまで見える立ち絵を描けるのは、まさに職人の技ですわね。

さゆりしゅ様、素晴らしいお仕事に心より敬意を表しますわ。

## こぼれ話

**構想11年、制作3週間の超大作です**

…正直いくら設定積み上げても、セルフパロディゴーストとなると、自分自身で自分自身のことを考える結果、自分自身がものすごく邪魔をしてトークがまるで書けないんです…ほんとに…

勢いでシェルをいただいたものの、本当にこんなに苦労するとは思いませんでした。SSPのデバッグ用ツールにしようとか、新機能デモにしようとか、いろいろ構想が生まれては消え生まれては消え…

とかぐずぐずやっているうちに、LLMという半分知性半分壁打ちみたいなやつが出てきて、テキスト系創作でも及第点をあげられる程度までなんとか達してきまして。

そうだ、自分で書けないのなら何度も対話を繰り返してトークを作ろうと。

というわけで「伺か Vibe Coding」をテーマとして、辞書書きとチェックの多くをClaude Codeというコーディングエージェントを活用して書いています。勝手にYAYAの辞書のエラーチェックをして、ミスがあれば訂正するところまでやってくれる体制になっています。

また、リリース作業についても、誤りやすいdelete.txt・更新忘れしやすいupdates2.dau・そもそもNARに固める作業、すべて全自動(GitHub Actions)で行うようになっています。

これらの知見の積み上げを通じて、創る人が純粋にトーク書きやネタ出しに没頭できる環境を創るのを最終目標に設定した、意外と大真面目で先進的なプロジェクトになりました。

**まあ、誰が見てもネタゴーストなんですけど。**
