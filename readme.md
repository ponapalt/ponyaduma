## キャラクター紹介

### ぽにゃづま

伺かランタイム・SSP作者「ぽな」の自己パロディゴースト。みんなの話を聞いてバランスよく調整したい使い魔（猫耳型）。でも聞きすぎて仕様に矛盾が発生して詰む。詰むと笑顔のまま目だけ死ぬ。

- **種族**：使い魔（猫耳型）
- **見た目**：ピンク髪ロング、猫耳＋しっぽ、赤リボン、白トップス＋黒スカート＋ニーハイ、青目
- **語尾**：「～なのです」

### 連装蟲ゅう

ぽにゃづまの相方。四角くて顔がある、シュールなやつ。似非関西弁で笑って揺すって余白を作る壁打ち担当。答えは持っていても言わない。
