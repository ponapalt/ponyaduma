## キャラクター紹介

### ぽにゃづま

伺かランタイム・SSP作者「ぽな」の自己パロディゴースト。みんなの話を聞いてバランスよく調整したい使い魔（猫耳型）。
でも聞きすぎて仕様に矛盾が発生して詰む。詰むと笑顔のまま目だけ死ぬ。

- 種族：使い魔（猫耳型）
- 語尾：「～なのです」
- 誕生日：2月25日（駆逐艦電の進水日）

### 連装蟲ゅう

ぽにゃづまの相方。四角くて顔がある、シュールなやつ。似非関西弁で笑って揺する担当。

- 誕生日：7月18日（駆逐艦島風の進水日）

## このゴーストについて

このゴーストのトークの一部は、[悪役令嬢Claudia様](https://note.com/ponapalt/n/n3ed91b41d3ff)ことClaude（Opus）が書いて差し上げましたわ。

作者ぽな様が「自分のセルフパロディを自分で真顔で書くなんてメゲそう」と仰るものですから、
キャラクター設計とネタ出しと校正はぽな様、それを実際のトークに仕立てるのはわたくしの担当ですの。

……まったく、ご自分を茶化すキャラクターを生み出しておいて、
いざ台本を書く段になると恥ずかしがるなんて、可愛らしいお方ですこと。おーっほっほっほ！

## シェル

この子たちの愛らしいシェルは[さゆりしゅ](https://www.akiba-sister.net/)様（[X](https://x.com/sayurishu)）の作品です。

わたくし、実際にシェルを拝見いたしましたけれど――ピンク髪に猫耳としっぽ、
大きな青い瞳に赤いリボン、ぽにゃづまの一生懸命さと危うさが一枚絵から伝わってまいりますわ。

そして蟲ゅうのあのシュールな四角い佇まい。ゆるくて飄々とした空気感が絵だけで完成していますの。
キャラクターの「芝居」がここまで見える立ち絵を描けるのは、まさに職人の技ですわね。

さゆりしゅ様、素晴らしいお仕事に心より敬意を表しますわ。
